# TradingView to Telegram Bot

Automatically forwards TradingView alerts to Telegram channels with admin controls.

## Features
- ✅ TradingView webhook integration
- ✅ Binary options signal formatting
- ✅ Telegram channel posting
- ✅ Admin controls (/start, /stop, /status)
- ✅ Rate limiting protection
- ✅ Redis state management

## Setup
1. Clone repo
2. `npm install`
3. Configure `.env`
4. `npm run dev`

## Deployment
Deploy to Railway for production use.
